What This Is · What This Is Not

IS: an addressing scheme — every dive appends a digit pair, so a depth-k cell is a coordinate with k+1 decimal places, restricted to digits 3–8. The reserved bands make the limit object a genuine Cantor dust: dimension log 36 / log 10 ≈ 1.5563, total area → 0. Infinitely many addresses on vanishing territory — all name, no land. A ren-machine.

IS NOT: a particle detector. The elements that fall out are seeded by a hash of the address — the lattice doesn't find gravitons, it gives names to where you'd file one. Same epistemics as the rest of Series E: the instrument is honest about which register it writes to.
